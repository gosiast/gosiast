# 🥤 What I'm working on
At **SODAX** I build and ship the production frontend for a cross-chain execution layer, working across a Turborepo + pnpm monorepo:
- 🔁 Product surfaces: swap, money market (lending), staking, pool / liquidity, and token migration flows
- 🔗 Multi-chain wallet integration across EVM, Solana, Sui, Stellar, and Injective (connect, sign, broadcast)
- 🧩 Next.js 15 (App Router) + React 19, Tailwind, and shadcn/ui, with Zustand and React Query for state
- 📈 Marketing & BD pages: partner directories, brand kit, press kit, and live on-chain stats integrations
- 🛠 Contributing across shared SDK and React hook packages that the app consumes
